How It Works

From bug detection to fix suggestion — fully automated.

1. PR Opened

Webhook fires when a pull request is opened or updated on your repository.

2. AI Detects Bugs

The engine analyzes changed files and identifies potential bugs using GPT-4o-mini.

3. Sandbox Validates

The generated fix runs in an isolated sandbox against the project's test suite.

4. You Approve

A "Commit suggestion" button appears on the PR. Review, click, and merge.
